# 作品说明表

## 基本信息
- 花名：陀飞轮
- 类别：运营类
- 作品名称：投资·AI产业链每日情报

## 使用工具
- AI工具：Claude Code

## 作品用途
作品是什么：本作品是一款聚焦 AI 产业链的每日情报聚合，核心功能是对 AI 产业链（算力与国产替代、AI 应用、物理 AI 三大核心赛道）的每日新闻、政策动向、研报解读、公司动态、投融资信息进行结构化整合与可视化呈现。
解决的核心问题：
1、信息碎片化：整合分散在不同平台的 AI 产业链信息，避免人工逐一筛选的低效性；
2、赛道关注度量化：通过 “赛道热度矩阵” 直观展示各 AI 细分赛道的每日信息产出量，快速判断赛道活跃度；
3、决策辅助：提炼每日投资建议、关键事件，为 AI 产业链相关投资决策、行业研究提供轻量化参考；
4、信息快速检索：支持按赛道、关键词筛选信息，提升目标信息获取效率。

与岗位业务的关联：
1、日常跟踪 AI 产业链动态，快速掌握各赛道热度、核心事件与头部公司动向，辅助投研报告撰写、投资标的筛选；
2、监测 AI 产业链各细分领域的信息密度与热点变化，分析行业发展趋势；
3、了解被投企业所在赛道的行业动态，及时捕捉政策、市场机会与风险点；
4、通过轻量化看板快速掌握 AI 产业链每日核心信息，无需深入阅读海量原始资料。

## 使用说明
操作步骤：
1、核心信息查看：打开页面后，默认展示当日 AI 产业链核心摘要、赛道热度矩阵、关键词热度；
2、点击 “赛道热度矩阵” 中的任意赛道卡片（如 “算力与国产替代”），自动跳转到该赛道的新闻列表；
3、在 “新闻流” 标签下，点击子标签（全部 / 算力与国产替代 / AI 应用 / 物理 AI）筛选对应赛道新闻；
4、关键词筛选：点击 “关键词热度” 中的任意关键词标签，自动跳转到对应赛道并高亮含该关键词的新闻；
5、多维度信息切换：通过页面中部的标签栏（新闻流 / 政策动向 / 研报解读 / 公司动态 / 投融资）切换不同类型的信息板块；
6、历史回顾：点击页面顶部 “📅 历史回顾” 按钮，在弹窗中选择历史日期，查看过往日期的 AI 产业链情报；
7、链接跳转：所有新闻 / 研报标题均为可点击链接，点击后跳转至原始信息源（新标签页打开）。
运行方式：环境要求：支持现代浏览器（Chrome/Firefox/Edge/Safari 等），无需额外安装软件]

适用人群：
1、金融机构 AI 产业链投研人员；
2、AI 行业创业 / 从业人士；
3、关注 AI 产业链的投资机构决策层、分析师；
4、高校 / 研究院所从事 AI 产业研究的人员；
5、其他需要快速掌握 AI 产业链每日动态的相关从业者。

## 创作过程
1、需求拆解：向 Claude Code 明确核心需求 —— 打造 AI 产业链每日情报看板，需包含信息聚合、可视化、多维度筛选等功能，同时明确 UI 风格（轻量化、专业级、适配移动端）；
2、代码生成：基于需求，Claude Code 完成核心代码框架搭建，包括：
HTML 结构：按 “头部 - 核心看板 - 多标签内容区 - 弹窗” 分层设计页面骨架；
CSS 样式：定义符合金融 / 产业分析场景的视觉风格（低饱和度配色、卡片化布局、响应式适配）；
JavaScript 逻辑：实现数据渲染、赛道 / 关键词筛选、标签切换、历史回顾弹窗等交互功能；
3、数据适配：提供 AI 产业链结构化数据模板后，Claude Code 优化数据渲染逻辑，确保不同维度数据（摘要 / 新闻 / 公司动态）能适配页面布局；
4、功能迭代：针对初始版本的交互细节（如关键词点击高亮、赛道卡片 hover 效果、移动端适配），通过自然语言指令反馈给 Claude Code，完成代码迭代优化；
5、最终调优：人工微调部分样式细节（如配色、间距），补充数据嵌入逻辑，完成成品。
